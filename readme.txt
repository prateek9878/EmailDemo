To execute the test cases there are 2 options.

Option 1:
1. Import the project in editor(Eclipse).
2. Configure the test classes in tentng.xml which you want to execute.
3. Build the pom.xml.

Option 2(without editor(Eclipse)):
1. Download the project in local folder.
2. Configure the test classes in tentng.xml which you want to execute.
3. Run the run.bat file.
